---
name: Writing style rules
description: No em dashes, write human-like text instead of typical AI style
type: feedback
---

1. **Keine Gedankenstriche (–) verwenden.** Stattdessen Kommas oder einfache Satzstrukturen nutzen.
   **Why:** User empfindet Gedankenstriche als typischen KI-Stil und will das nicht.
   **How to apply:** Bei allen Texten, Headlines, Subtexts und Copy auf der Website.

2. **Menschlicher schreiben, nicht wie eine KI.** Natürlicher Ton, keine übertrieben polierten Formulierungen.
   **Why:** Der User will authentische, nahbare Texte statt glattgebügelter KI-Sprache.
   **How to apply:** Beim Schreiben und Korrigieren aller Website-Texte. Lieber einfach und direkt als fancy.
